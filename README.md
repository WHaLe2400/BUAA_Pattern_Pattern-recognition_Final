__**UpDate25.6.16**__:  
Main.0.5程序对于加载数据的部分进行了修改，这样数据会在开始训练前全部保存在RAM中，再训练过程中会显著加快训练速度（前提是GPU跟得上）可以重复你运用到GPU性能，<br />
需要注意的是，这一操作需要占用约5G的内存空间（如果不加载test文件夹则需要3.5G），用的时候注意自己的内存分配。
