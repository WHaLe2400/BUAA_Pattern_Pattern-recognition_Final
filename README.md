__**UpDate25.6.16**__:  
Main.0.5程序对于加载数据的部分进行了修改，这样数据会在开始训练前全部保存在RAM中，在训练过程中会显著加快训练速度（前提是GPU跟得上）可以充分你运用到GPU性能，<br />
